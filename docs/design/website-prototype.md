# MyCoffeeStore 网站原型设计文档

## 设计概述

### 品牌定位
- **品牌名称**: MyCoffeeStore
- **位置**: 旧金山 Haight Ashbury
- **定位**: 精品咖啡店
- **标语**: "每一杯咖啡，都是一种生活方式"

### 设计风格
- **色调**:
  - 主色: `#1A1A1A` (深黑) - 专业、高端
  - 强调色: `#D4A574` (金棕色) - 咖啡色、温暖
  - 背景色: `#FAFAFA` (米白) - 干净
  - 辅助色: `#CCCCCC` - 次要文字

- **字体**:
  - 标题: Playfair Display (优雅衬线)
  - 正文: 系统默认无衬线

- **设计语言**:
  - 简约现代
  - 温暖舒适
  - 高端精品

## 响应式设计

### 断点
- Mobile: < 768px
- Tablet: 768px - 1024px
- Desktop: > 1024px

### 移动端适配
- 导航栏: 汉堡菜单
- 网格: 1列布局
- 卡片: 全宽 + 底部圆角
- Hero: 居中文字，移除大图背景
