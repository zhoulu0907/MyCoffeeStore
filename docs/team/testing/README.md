## 测试命令参考

### 前端测试
```bash
# 单元测试
npm run test

# 测试覆盖率
npm run test:coverage

# E2E 测试
npm run test:e2e

# E2E 测试 UI 模式
npm run test:e2e:ui
```

### 后端测试
```bash
# 所有测试
mvn test

# 特定测试类
mvn test -Dtest=UserServiceTest

# 覆盖率报告
mvn jacoco:report
```

### 测试数据
```bash
# 导入测试数据
mysql -u root -p my_coffee_store < docs/team/testing/测试数据.sql
```
